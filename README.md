# Document Converter & PDF Merger

Aplicación que permite convertir documentos de Word (.docx) a formato PDF y unir múltiples archivos PDF en un solo documento.

La herramienta está diseñada para facilitar la gestión de documentos, ofreciendo una forma rápida y sencilla de generar PDFs a partir de archivos Word y combinar varios PDFs en un único archivo final.

Ideal para uso académico, administrativo o personal.
